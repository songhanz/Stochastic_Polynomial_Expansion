# Pseudo-Taylor Polynomial Expansion (PTPE)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Published in TMLR](https://img.shields.io/badge/Published%20in-TMLR%20(06%2F2025)-blue)](https://openreview.net/forum?id=lyDRBhUjhv)

Official code for **["A Stochastic Polynomial Expansion for Uncertainty Propagation through Networks"](https://openreview.net/forum?id=lyDRBhUjhv)**, published in Transactions on Machine Learning Research (TMLR), 06/2025.

**Authors:** Songhan Zhang, ShiNung Ching (Washington University in St. Louis)

PTPE propagates multivariate Gaussian distributions through neural network nonlinearities using closed-form polynomial expansions, providing accurate uncertainty estimates at O(n²) complexity without Monte Carlo sampling. It supports seven activations: Tanh, Sigmoid, Softplus, ReLU, LeakyReLU, GELU, and SiLU.

This method extends our earlier work on stochastic linearization of neural networks:

> S. Zhang, M. Singh, D. Menolascino, S. Ching. *Estimating Uncertainty from Feed-Forward Network Based Sensing Using Quasi-Linear Approximation.* Neural Networks, Vol. 188, 2025. [[Paper]](https://doi.org/10.1016/j.neunet.2025.107376) [[Code]](https://github.com/songhanz/Stochastic-Linearization-of-Neural-Nets)

Stochastic linearization corresponds to the first-order (S=1) case of PTPE. Higher-order terms (2nd, 3rd) capture curvature effects that linearization misses, dramatically improving accuracy under moderate-to-high input uncertainty.


## Setup

**DVI experiments** (TensorFlow-based):
```bash
cd DVI+PTPE
pip install -r requirements.txt    # TensorFlow 1.15, TF-Probability 0.5, scikit-learn, etc.
```

**VAE experiment** (PyTorch-based):
```bash
pip install torch torchvision numpy scipy matplotlib pandas
```

**CIFAR-10 experiment**: MATLAB R2021a+ with Deep Learning Toolbox and GPU support.

## Quick Start

### DVI+PTPE — UCI Regression (Tables 2 & 3)

Trains Bayesian neural networks with PTPE on 8 UCI datasets (auto-downloaded). One script per activation:

```bash
cd DVI+PTPE
python UCI_relu.py     # ReLU activation
python UCI_gelu.py     # GELU activation
python UCI_tanh.py     # Tanh activation
```

Results are saved to `UCI_results_sgd/<dataset>_<activation>_<layers>layer_adapter/`. To print aggregated RMSE and log-likelihood:

```bash
python print_result.py
```

Also see `ToyData.ipynb` for an interactive 1D demo.

### DVI+PTPE — MNIST Classification & OOD Prep

```bash
cd DVI+PTPE_categorical
python mnist.py                # Train BNN on MNIST → saves to mnist_dvi_model/
python rotation_ood.py         # Evaluate on rotated MNIST & FashionMNIST OOD
                               # → saves .pkl results to mnist_dvi_results/
```

Optional variants: `mnist_search_lambda.py` (λ grid search), `mnist_cyclic_anneal.py` (cyclic KL annealing).

### VAE+PTPE — Variational Autoencoder (Figure 5)

Trains a vanilla VAE and a PTPE-VAE side by side on MNIST. Set `download=True` in the script on first run.

```bash
cd VAE+PTPE
python train_mnist.py              # Trains both models, saves ELBO curves as CSV/PNG
python reconst_err_vanilla.py      # Reconstruction MSE for vanilla VAE
python reconst_err_PTPE.py         # Reconstruction MSE for PTPE-VAE
```

Edit the `for latent_size in [2]:` loop in `train_mnist.py` to `[2, 5, 10, 20]` for all latent dimensions. Publication figures are generated by `make_figures.m` in MATLAB.

### OOD Detection — Rotated MNIST (Figure 4)

Compares DVI+PTPE against Vanilla, Dropout, and Ensemble baselines. Requires DVI results from the previous step:

```bash
# Copy DVI predictions into this folder
cp -r DVI+PTPE_categorical/mnist_dvi_results/ rotation_mnist_ood/mnist_dvi_results/

cd rotation_mnist_ood
python main.py    # Trains baselines, evaluates all models on rotated MNIST & FashionMNIST OOD
                  # → results/uncertainty_estimation_results.pdf
```

### CIFAR-10 — ResNet Uncertainty Propagation (Figures 3, 9, 10)

All MATLAB scripts. Requires pretrained ResNet `.mat` files in a `models/` directory.

```matlab
% Step 1: Flatten convolution weights (run once)
flatten_resnet

% Step 2: Monte Carlo ground truth
true_output_dist_Tanh    % (also: true_output_dist_ReLU, true_output_dist_GELU)

% Step 3: PTPE estimation at orders 1–3
est_output_PTPE_3orders_resnet_tanh   % (also: _relu, _gelu, _tanh_8gamma)

% Step 4: Generate comparison figures
compare_PTPE_JL
compare_PTPE_JL_werrorbar
```
